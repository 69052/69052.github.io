hexo博客搭建过程记录

第一步 安装环境

  • 1.安装Node.js
  • 2.安装Git

第二步 生成博文

安装npm

  • windows，cmd命令行

  • 安装npm
    $ npm install -g hexo-cli
    可以使用淘宝镜像：
    $ npm install -g hexo-cli –registry=https://registry.npm.taobao.org

  • 初始化命令：
    cd到某个路径，进行初始化
    hexo init

windows操作tips

  • 先输入盘符名称，如E: 再使用cd命令进行跳转

  • 新建文章命令
    $ hexo new [layout] <br>新建一篇文章。如果没有设置 layout 的话，默认使用_config.yml 中的 default_layout 参数代替。如果标题包含空格的话，请使用引号括起来。

  • 生成命令
    hexo generate 或者 hexo g

  • 本地服务预览命令
    $ hexo s

第三步 推送到github

  • 1.要使用远程部署需要先运行以下命令安装hexo-deployer-git
    npm install hexo-deployer-git –save

  • 2.Github创建新仓库，仓库名称要和用户名一致，以io结尾
    例如：用户名 JoLangldy 仓库名 JoLangldy.github.io

  • 3.博客路径下找到_config.yaml文件，修改此部分

deploy:
type: git
repo: https://github.com/JoLangldy/JoLangldy.github.io.git
branch: master

  • 4.推送命令：
    hexo d
